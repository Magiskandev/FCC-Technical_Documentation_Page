Introducción

En esencia, HTML es un lenguaje bastante simple compuesto por elementos, que se puede aplicar a fragmentos de texto para darles un significado diferente en un documento, estructurar un documento en secciones lógicas e incrustar contenido como imágenes y videos en una página. Antes de comenzar, no necesita ningún conocimiento previo de HTML, pero debe tener al menos una familiaridad básica con el uso de computadoras y la web de forma pasiva.

  • Por ejemplo, solo mirarlo y consumir contenido en la web.
  • Debe tener configurado un entorno de trabajo básico (navegador web y editor de texto).
  • También comprende cómo crear y administrar archivos.

CSS (Cascading Style Sheets / hojas de estilo en cascada) se utiliza para diseñar y estilizar páginas web, por ejemplo, para modificar la fuente, el color, el tamaño y el espaciado de su contenido, dividirlo en varias columnas o agregar animaciones y otras características decorativas. Antes de comenzar, debe tener:

  • Familiaridad básica con el uso de computadoras y el uso pasivo de la Web.
  • Un entorno de trabajo básico configurado como se detalla en Instalación de software básico y conocimientos sobre cómo crear y administrar archivos.
  • Familiaridad básica con HTML.
Introducción a HTML

¿Qué es HTML?

HTML (Hypertext Markup Language / lenguaje de marcado de hipertexto) no es un lenguaje de programación; es un lenguaje de marcado que se utiliza para indicarle a su navegador cómo estructurar las páginas web que visita. Puede ser tan complicado o tan simple como lo desee el desarrollador web. HTML consta de una serie de elementos que se utilizan para encerrar, ajustar o marcar diferentes partes del contenido para que aparezca o actúe de cierta manera. Las etiquetas adjuntas pueden convertir un poco de contenido en un hipervínculo para vincularlo a otra página en la web, poner palabras en cursiva, etc. Por ejemplo, tome la siguiente línea de contenido:
Mi nombre es Alejandro Osorio Velasco.
<p>Mi nombre es Alejandro Osorio Velasco.</p>

** Nota: Las etiquetas en HTML no distinguen entre mayúsculas y minúsculas, es decir, pueden escribirse en mayúsculas o minúsculas. Por ejemplo, una etiqueta <title> podría escribirse como <title>, <TITLE>, <Title>, <TiTlE>, etc. y funcionará bien. La mejor práctica, sin embargo, es escribir todas las etiquetas en minúsculas por razones de coherencia, legibilidad por otras razones. **

Introducción a CSS

¿Qué es CSS?

CSS (Cascading Style Sheets / hojas de estilo en cascada) es un lenguaje para especificar cómo se presentan los documentos a los usuarios - cómo se diseñan, distribuyen, etc. Le permite crear páginas web de gran apariencia, pero ¿cómo funciona bajo el capó? ¿Cómo convierte el navegador HTML en un modelo de objetos de documento (DOM), ¿cómo se aplica CSS a partes del DOM y qué código se utiliza para incluir realmente nuestro CSS en nuestra página web?
A CSS stylesheet is applied to the HTML using a <link> element within the <head> tag. For example: Se aplica una hoja de estilo CSS al HTML mediante un elemento <link> dentro de la etiqueta <head>. Por ejemplo: <link rel="stylesheet" href="style.css">

¿Cómo afecta CSS a HTML?

Los navegadores web aplican reglas CSS a un documento para afectar cómo se muestran. Un conjunto de reglas CSS contenidas en una hoja de estilo determina cómo debe verse una página web. Una regla CSS se forma a partir de:

  • Un conjunto de propiedades, las cuales tienen valores configurados para actualizar el contenido HTML mostrado, por ejemplo, quiero que el ancho de mi elemento sea el 50% de su elemento principal y que su fondo sea rojo.
  • Un selector, que selecciona los elementos a los que desea aplicar los valores de la propiedad actualizados. Por ejemplo, quiero aplicar mi regla CSS a todos los párrafos de mi documento HTML.
Sintaxis de HTML

La anatomy de un elemento HTML

Vamos a explorar el elemento parrafo un pocoLets explore the paragraph element a bit... <p>Mi perro es muy fuerte.</p>

Las principales partes de este elemento son:

  1. La etiqueta de apertura: Consiste en el nombre del elemento (en este caso, p), envuelto entre paréntesis angulares de apertura y cierre. Esto indica dónde comienza o empieza a surtir efecto el elemento, en este caso, dónde está el comienzo del párrafo.
  2. La etiqueta de cierre: es la misma que la etiqueta de apertura, excepto que incluye una barra inclinada antes del nombre del elemento. Esto indica dónde termina el elemento, en este caso, dónde está el final del párrafo. No incluir una etiqueta de cierre es un error común para principiantes y puede dar lugar a resultados extraños.
  3. El contenido: este es el contenido del elemento, que en este caso es solo texto.
  4. El elemento: la etiqueta de apertura más la etiqueta de cierre más el contenido es igual al elemento.
Sintaxis CSS

La anatomía de una propiedad de CSS

En primer lugar, tomemos un documento HTML simple, que contiene una <h1> y una <p> (observe que se aplica una hoja de estilo al HTML mediante un elemento):
<!DOCTYPE html> <html> <head> <meta charset="utf-8"> <title>Mi prueba con CSS</title> <link rel="stylesheet" href="style.css"> </head> <body> <h1<¡Hola Mundo!</h1> <p>Este es mi primer ejemplo de CSS</p> </body> </html>

Ahora veamos un ejemplo de CSS muy simple que contiene dos reglas: h1 { color: blue; background-color: yellow; border: 1px solid black; } p { color: red; }

La primera regla comienza con un selector h1, lo que significa que aplicará sus valores de propiedad al elemento <h1>. Contiene tres propiedades y sus valores (cada par de propiedad / valor se llama declaración):

  1. El primero establece el color del texto en azul.
  2. El segundo establece el color de fondo en amarillo.
  3. El tercero coloca un borde alrededor del encabezado de 1 píxel de ancho, sólido (no punteado, discontinuo, etc.) y de color negro.

La segunda regla comienza con un selector p, lo que significa que aplicará sus valores de propiedad al elemento <p>. Contiene una declaración, que establece el color del texto en rojo.
